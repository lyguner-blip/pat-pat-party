# Changelog

## 0.0.6 - 队友目标互动与互动菜单

- 修复普通玩家不拥有队友 Token 时难以互相互动的问题：当自己的 Token HUD 中只有一个其他 Token 被 target 时，摸头、抱抱、碰拳、送花和递茶会优先作用到该 target Token。
- 新增送花互动：Dialog、自定义文案、聊天卡片、PIXI/Canvas 动画、DOM 降级和 socket 同步。
- 送花动画支持目标 Token 头顶花束、小花、爱心和星星粒子，并复用现有校准偏移。
- 将摸头、抱抱、碰拳、送花和递茶集成到一个 Token HUD “互动”按钮中，减少 HUD 按钮拥挤。
- 新增碰拳互动：双 Token 小拳头从两侧对撞，中心爆出冲击圈、星星和鼓气粒子，适合接力、鼓气或庆贺。
- 新增递茶互动：茶杯、热气、绿叶粒子、聊天卡片、socket 同步和 DOM 降级。
- 修复部分 Foundry 主题下聊天卡片文字颜色被继承样式压淡的问题。
- 新增 `cn` 中文语言包注册，兼容使用 `cn` 语言代码的 Foundry 中文汉化补丁，同时保留 `zh-CN`。
- 保持敌对 Token 限制、玩家互摸设置、冷却、GM 权限和摸头三种力度不变。

## 0.0.5 - 抱抱互动基础版

- 新增 Token HUD “抱抱”按钮，可由已选中/归属自己的 Token 对另一个 Token 发起抱抱。
- 新增抱抱 Dialog，支持自定义一句显示文案。
- 新增抱抱 PIXI/Canvas 动画：两个 Token 中间出现贴贴弧线、抱抱表情、爱心和粒子。
- 抱抱互动复用现有权限、冷却、聊天开关与 socket 同步流程。
- 新增抱抱聊天卡片与中英文本地化。
- 保留摸摸头三种力度、头顶校准、冷却、GM 设置和 socket 同步不变。

## 0.0.4 - PIXI 摸头动画优化

- 摸头动画改为优先使用 PIXI/Canvas 临时容器播放，并保留 DOM overlay 降级路径。
- 动画锚点改为 Token 头顶 world 坐标，并继续应用 `patOffset` 校准偏移。
- 小手从右上滑入，在头顶左右揉搓并轻微旋转，结束时上浮淡出。
- Token 视觉层增加压缩、回弹或晃动反馈，结束后恢复原始状态。
- 爱心、小花、星星粒子分批冒出，具备随机大小、漂移、缩放、旋转和淡出。
- 大力逆毛摸增加短弧线和星星表现乱毛感，但保持软萌互动风格。
- 清理动画结束后的 PIXI ticker、临时容器和 Token mesh 反馈，避免残留。

## 0.0.3 - 自定义亲密度

- 修复 Foundry manifest 安装缺少 `url`、`manifest`、`download` 字段的问题。
- 新增自定义摸头文案，默认仍为“亲密度上升↑”。
- 自定义文案会同步显示在 Token 上方浮动文字和聊天卡片中。
- 优化三种力度动画：小手从右上滑入、左右揉搓、上浮淡出，并增加 Token 轻反馈。
- 爱心、小花、星星粒子改为分批随机漂浮，按力度改变数量和幅度。
- Socket 同步 payload 增加自定义文案与校准偏移。
- 记录抱抱互动需求：选中队友 Token → 抱抱 → 两个 Token 之间出现贴贴/爱心动画 → 聊天栏提示温暖文案。

## 0.0.2 - 亲密度上升

- 新增 Token 头顶位置手动校准。
- 摸头动画改为小手左右滑动，并在旁边冒爱心。
- 聊天文案改为固定“亲密度上升↑”。
- 新增三种摸头力度：轻柔摸、普通摸、大力逆毛摸。
- 为三种力度分别提供不同动画表现。

## 0.0.1 - MVP

- 初始版本：Token HUD 摸摸头按钮、权限判断、冷却、DOM 动画、socket 同步与本地化聊天卡片。
